Image Converter using Python, Flask, and Bootstrap

This project is an Image Converter web application developed using Python, Flask, and Bootstrap. The application allows users to upload images and perform basic image processing tasks such as converting them to grayscale, png, jpeg , and pdf.

Key Features:

Image Upload: Users can upload images in various formats, such as PNG, JPG, JPEG, GIF, and WEBP
Image Processing: The application processes the uploaded images based on user-selected operations, such as converting images to grayscale using OpenCV.
Responsive UI: The frontend is developed using Bootstrap, ensuring a responsive and user-friendly interface across different devices.
Downloadable Output: After processing, the converted image is saved on the server and can be downloaded by the user.

Technology Stack:

Python: Core programming language used for backend development.
Flask: Lightweight web framework used to create routes and handle requests and responses.
OpenCV: Python library used for image processing tasks.
Bootstrap: CSS framework for building responsive and visually appealing UI.
HTML/CSS: For structuring and styling the frontend pages.

User Flow:

Home Page: A simple interface with options to upload an image and choose the desired operation. Image Upload: Users can upload an image, and the application checks for allowed file types. Image Processing: The uploaded image is processed according to the selected operation (e.g., grayscale conversion). Output and Download: The processed image is saved in the server's static folder, and the user is provided with a link to download the result.
